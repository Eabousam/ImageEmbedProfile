### Image Analysis and ProvGigaPath Embedding Vignettes

| File | Description |
|------|-------------|
| `run_mofa.rmd` | **MOFA Training** – Loads Prov‑GigaPath slide embeddings and aligns them with TCGA molecular assays (RNA‑seq, RPPA, CNV) for COAD and OV. Creates patient‑level matrices, trains MOFA models (15 factors), and visualises variance explained per factor and per view. Outputs trained `.hdf5` models for downstream analysis. |
| `embed_analysis.qmd` | **Direct Embedding Interpretation** – Analyses individual Prov‑GigaPath embedding dimensions without MOFA. Identifies most/least variable features, runs univariate linear regression (for stage) and Cox regression (for survival) per embedding, computes GSVA pathway scores, and correlates embeddings with Hallmark pathways. Generates cross‑cancer heatmaps and top‑association tables for KIRC, COAD, LUAD, OV, and READ. |
| `mofa_analysis.qmd` | **MOFA Downstream Analysis** – Loads trained MOFA models and performs comprehensive biological interpretation: FGSEA on RNA‑seq factor weights (Hallmark, C2, C6), GSVA sample‑level pathway scores, correlation of factors with pathway activities, survival analysis (Kaplan‑Meier, univariate/multivariable Cox), and embedding‑factor projections. Generates interactive HTML summary reports with factor‑by‑factor pathway enrichments and embedding correlations. |
| `run_mofa_gsea.qmd` | **MOFA + GSEA Integration** – Builds on MOFA output to run preranked GSEA (FGSEA) on factor weights using Hallmark, C2, and C6 gene sets. Produces enrichment tables and visualisations to link each factor to biological processes. |
| `analysis_tile_level.qmd` | **Tile‑Level Embedding Analysis** – Explores Prov‑GigaPath embeddings at the individual tile level rather than slide level. Enables spatial mapping of embedding features to tissue regions, allowing investigation of local morphological patterns and their associations with molecular profiles. |
| `cancer_image_Hovernet_report_1.qmd` | **HoVer‑Net Cell Segmentation** – Integrates HoVer‑Net outputs (cell segmentation, nuclear features) with pathology embeddings and multi‑omics data. Provides a framework to connect cellular‑level features (nuclear morphology, spatial arrangement) to slide‑level embeddings and molecular outcomes. |
